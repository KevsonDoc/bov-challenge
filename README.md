## Como baixar o projeto
```bash
    #Clonar repositório
    $ git clone git@github.com:KevsonDoc/bov-challenge.git

    #Acessar repositório
    $ cd bov-challenge
```
---
## Instalando programas e dependência para executar a aplicação.
```bash
#Instalando dependências necessárias para aplicação
$ yarn install

    ou 

$ docker compose up
```

---

### Iniciando aplicação

```bash
    $ yarn start:dev
```
